# PyClimaExplorer 🌍
An interactive dashboard for visualizing global climate prospects: **Temperature, Wind Flow, and Precipitation.**

## 🛠️ Built With
- **Streamlit**: Web Interface
- **Xarray**: Multi-dimensional data handling
- **Plotly**: Interactive mapping and graphing

## 🚀 How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
